# ProjetoAgrinho2026 
Este site foi desenvolvido como parte do Projeto Agrinho, com objetivo de apresentar a importancia e as caracteristicas do agro no Sudoeste do Paraná, destacando a agricultura, a sustentabilidade, a preservação ambiental e as tecnologias utilizadas no campo de forma criativa e interativa.

O projeto tambem serve como prática de programação WEB utilizando linguagem HTML, CSS e JavaScript.

# Funcionalidades
- Layout moderno responsivo;
- Galeria de imagens do agro;
- Cards informativos;
- Navegação suave;
- Menu de acessibilidade;
- Efeitos visuais;
- Quiz interativo.
